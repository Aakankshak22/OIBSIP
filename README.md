## 3. Project Details

### Iris Flower Classification

- **Description**: In this project, I delved into the world of supervised machine learning. I used the famous Iris dataset to build a classification model that can identify and categorize iris flowers into three different species based on their sepal and petal characteristics.

- **Techniques Learned**: Data preprocessing, feature selection, model selection, and evaluation.

- **Skills Gained**: Understanding of classification algorithms (e.g., Decision Trees, K-Nearest Neighbors), data visualization, and model evaluation techniques.

### Unemployment Analysis with Python

- **Description**: This project revolved around data analysis using Python. I explored datasets related to unemployment rates over time and aimed to derive meaningful insights. I employed data visualization, statistical analysis, and time series analysis to gain a deeper understanding of unemployment trends.

- **Techniques Learned**: Data cleaning, data visualization with libraries like Matplotlib and Seaborn, statistical analysis, and time series analysis.

- **Skills Gained**: Proficiency in data manipulation with Pandas, data visualization, and interpreting statistical results.

### Email Spam Detection with Machine Learning

- **Description**: My final project was focused on machine learning for text classification. I developed a spam detection system that could classify emails as spam or not spam (ham). I used natural language processing techniques, feature engineering, and machine learning algorithms to build an effective spam filter.

- **Techniques Learned**: Text preprocessing, feature extraction from text data, TF-IDF vectorization, and training machine learning models (e.g., Naive Bayes, SVM) for text classification.

- **Skills Gained**: Text data preprocessing, feature engineering, model training for text classification, and working with unstructured text data.
